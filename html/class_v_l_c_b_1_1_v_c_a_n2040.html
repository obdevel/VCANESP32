This function allows the PIO that is used to be specified. There are two PIOs in the RP2040, identified as PIO0 and PIO1) and thre in the RP253x, identified as PIO0, PIO1 and PIO2. If this function is not called, the default PIO is 0.

There are two buffer queues, one for transmit and one for receive. These buffers pace the message rate so as not to overwhelm slower processes. The larger the buffer, the more memory it uses. This function allows the user to specify the size of the buffer. The default size is 32 for transmit and 32 for receive.
